## 配置说明
### 身份验证配置
- 令牌过期时间：2小时

### Servlet配置
| Servlet名称        | 功能描述                              |
|--------------------|---------------------------------------|
| LoginServlet       | 处理用户登录请求，生成并返回认证令牌   |
| LogoutServlet      | 处理用户登出请求，使当前令牌失效       |
| ProfileServlet     | 处理用户个人信息的获取和更新操作       |
| StudentServlet     | 处理学生信息的注册、查询和管理操作     |
